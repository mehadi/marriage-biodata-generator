# Marriage Bio Data Maker

Create and export marriage biodata (personal, religious, education, professional, family info) with live preview. Supports drafts, templates, and PDF/image export. English & Bengali.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Use **Create** to build a biodata and export it.

## Build

```bash
npm run build
npm start
```

## Stack

Next.js 16, React 19, TypeScript, Tailwind CSS, React Hook Form, Zod, jsPDF, html-to-image.
